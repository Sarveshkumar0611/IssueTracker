
# Issuse Tracker

This is a nodejs + ejs  application to track issues/bugs for a project.

## Steps to start the project

1-Clone this project.           
2-Start by installing npm if you don't have it already.  
3-Navigate to Project Directory.

```bash
    npm install
    npm start

```
